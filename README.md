# homebrew-tap

Homebrew formulae for [quanghuynt14](https://github.com/quanghuynt14)'s tools.

## Install

```bash
brew install quanghuynt14/tap/bullmq-dash
```

Or tap first, then install:

```bash
brew tap quanghuynt14/tap
brew install bullmq-dash
```

## Formulae

| Formula                                                       | Description                                        |
| ------------------------------------------------------------- | -------------------------------------------------- |
| [bullmq-dash](https://github.com/quanghuynt14/bullmq-dash)    | Terminal and browser dashboard for BullMQ queue monitoring |
